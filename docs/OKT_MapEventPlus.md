# OKT_MapEventPlus

イベント制御・名札表示・親子連動などを拡張する多機能プラグイン。  
Cross Page Runner（ページ跨ぎ実行）、名札表示、親子イベント追従の統合版です。

---

## 導入方法
1. このリポジトリの `plugins/OKT_MapEventPlus.js` をダウンロード  
2. プロジェクトの `js/plugins/` にコピー  
3. プラグイン管理で有効化  

---

## 機能概要

### 名札（NameTag）表示
イベントの頭上に名前を表示できます。  
NPC名、看板、扉の行き先などに使用できます。

#### メモ欄タグ
```
<名前:村人A>
<Door:城下町の通路>
<NamePreset:Door>
```
#### プラグインコマンド
| コマンド名 | 説明 |
|-------------|------|
| 名札を表示（上書き） | 指定イベントの名札を即時表示（メモ欄より優先） |
| 名札を非表示 | 名札を強制的に隠す（再表示可能） |

---

### ページ跨ぎジャンプ／コール（Cross Page Runner）
イベントの別ページにある「ラベル」へ直接ジャンプまたはコールできます。  
ジャンプ後に元のページへ戻るかどうかを選択可能です。

- `CrossPageJump` … 戻らない  
- `CrossPageCall` … 呼び出し元に戻る  

---

### 親子イベント追従
メモ欄で親イベントを指定すると、子イベントが同じ動きをします。

```
<Parent:12>       イベントID12を親にする
<Parent[2]:12>    スイッチ2がONのときだけ追従
```

#### 特徴
- 親と同じ速度・頻度で移動  
- ジャンプや locate にも対応  
- 親イベントが停止中は子も停止  

---

### 歩行グラフィック描画位置調整
メモ欄で x軸・y軸 のオフセットを指定して、歩行グラフィックの表示位置を微調整できます。

```
<locate:x +3, y -8>
```
