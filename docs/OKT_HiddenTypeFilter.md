# OKT_HiddenTypeFilter

武器・防具・スキルタイプをメニューやUIから非表示にするプラグイン。  
隠しアイテムや隠しスキルを実装したいときに便利です。

## 導入方法
1. `plugins/OKT_HiddenTypeFilter.js` をダウンロード
2. プロジェクトの `js/plugins/` にコピー
3. プラグイン管理から有効化

## プラグインパラメータ
- **非表示タグ**: `<HideType>` を名前に含めると自動非表示
- **非表示スキルタイプID / 武器タイプID / 防具タイプID**
- **無効化スイッチID**

## 注意事項
- 初期装備やイベントコマンドで装備した場合、装備中リストには表示されます
- タグ判定とIDリスト判定の両方で非表示を行います
