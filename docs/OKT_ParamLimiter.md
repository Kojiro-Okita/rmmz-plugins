# OKT_ParamLimiter

標準パラメータに上下限とリミットブレイク用の上限を設定し、装備の `<LimitBreak>` タグで上限突破させる能力値を指定できる RPGツクールMZ 用プラグインです。

## 対応バージョン

- RPGツクールMZ

## 導入方法

1. `OKT_ParamLimiter.js` をプロジェクトの `js/plugins` フォルダに配置します。
2. プラグインマネージャーで **OKT_ParamLimiter** を有効にします。
3. 必要に応じてプラグインパラメータで各パラメータの  
   `min / max / breakMin / breakMax` を設定してください。

## 使い方（基本）

- 標準能力値（最大HP / 最大MP / 攻撃力 / 防御力 / 魔攻 / 魔防 / 敏捷 / 運）の
  最終値に対して上下限とブレイク上限を適用します。
- 上限突破させたい能力値は、装備のメモ欄に `<LimitBreak>` タグを記述して指定します。

### タグ

- 装備のメモ欄  
  - `<LimitBreak>`  
    - すべての標準パラメータをリミットブレイク対象にします。
  - `<LimitBreak[2]>`  
    - 指定した paramId（例: 2 = 攻撃力）のみブレイクさせます。
  - `<LimitBreak[0,2,4]>`  
    - 複数の paramId をカンマ区切りで指定できます。

※ paramId の対応は MZ の標準どおりです（0:最大HP, 1:最大MP, 2:攻撃力, ...）。

## 注意事項

- 本プラグインは **標準パラメータのみ** を対象とします。
- 拡張パラメータへの上下限・ブレイク適用を行いたい場合は、  
  別途有料プラグイン **OKT_AddParam** との併用を検討してください。
