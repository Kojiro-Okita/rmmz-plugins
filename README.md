# OKT_HiddenTypeFilter

RPGツクールMZ用プラグイン  
指定した武器タイプ・防具タイプ・スキルタイプをメニューやUIから非表示にできます。  
隠しアイテムや隠しスキルを実装したいときに便利です。

---

## 導入方法
1. このリポジトリの `OKT_HiddenTypeFilter.js` をダウンロード  
2. プロジェクトの `js/plugins/` フォルダにコピー  
3. プラグイン管理から **OKT_HiddenTypeFilter** を有効化  

---

## プラグインパラメータ
- **非表示タグ**  
  タイプ名にこの文字列を含めると自動的に非表示になります  
  例：`<HideType>`  
- **非表示スキルタイプID**  
  メニューから隠したいスキルタイプのIDを配列で指定  
- **非表示武器タイプID**  
  メニューから隠したい武器タイプのIDを配列で指定  
- **非表示防具タイプID**  
  メニューから隠したい防具タイプのIDを配列で指定  
- **無効化スイッチID**  
  指定したスイッチがONのときはプラグインが無効化されます（デバッグ用）

---

## 注意事項
- 初期装備やイベントコマンドで装備した場合、装備中リストには表示されます  
- タグによる非表示とIDリストによる非表示の両方で判定します
